In this demo, a chatbot answers 10 questions on specific client data about their rental lease. The pdf contract is ingested using python automation by scraping relevant CONTEXT INFORMATION. The CONTEXT INFORMATION is then passed to vector store and this is the only relevant context the model can answer about.

  • Python automation is used to scrape relevant data efficiently and effectively.
  • Users can only input 10 questions per session.
  • The text " Only respond regarding information related to the CONTEXT INFORMATION." protects the API from being used for purposes beyond the specific use case.
  • The data is scraped from a pdf contract, and does not connect to the appfolio API in order to maintain some control of the accessibility of data and make it less platform reliant. This drives costs down and allows more freedom. It can be cross checked with data from appfolio, but that is outside the scope of this demo.
