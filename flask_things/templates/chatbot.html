In this demo, a contract in the form of a pdf is ingested and python automation scrapes the file for relevant CONTEXT INFORMATION.

Showcase: examples of AI and automation (there can be more)

  • Chatbot uses openAI's API. Specifically:
    langchain and llama-index with the "text-davinci-003" model.
  • Python automation is used to scrape relevant data efficiently and effectively.

Showcase: examples of 2 security implementations (there can be more)

  • Users can only input 10 questions per session.
  • The text " Only respond regarding information related to the CONTEXT INFORMATION." protects the API from being used for purposes beyond the specific use case INFORMATION.
